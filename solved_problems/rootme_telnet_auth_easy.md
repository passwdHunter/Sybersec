# Telnet Authentication Analysis (RootMe)

**Платформа:** RootMe  
**Категория:** Network  
**Сложность:** Легкая  

##  Описание задачи
Анализ дампа сетевого трафика (PCAP) Telnet сессии для извлечения учетных данных аутентификации.

##  Процесс решения

### 1. Анализ файла
Получил файл содержащий захваченный Telnet трафик.

### 2. Открытие в Wireshark
Загрузил файл в Wireshark и применил фильтр для отображения только Telnet пакетов:
telnet

### 3. Реконструкция сессии
Использовал функцию "Follow TCP Stream" для восстановления диалога между клиентом и сервером:
- Правый клик на любом Telnet пакете
- Follow → TCP Stream

### 4. Извлечение учетных данных
В реконструированном потоке четко видны:
- Запрос имени пользователя: "login:"
- Запрос пароля: "Password:"
- Передаваемые в открытом виде учетные данные

##  Ключевые уроки
- **Telnet передает данные в открытом виде** - отсутствие шифрования
- **"Follow TCP Stream"** - мощный инструмент анализа сетевых протоколов
- **Анализ PCAP файлов** - базовый навык в сетевой безопасности
- **Небезопасные протоколы** не должны использоваться в production средах

##  Использованные инструменты
- Wireshark для анализа сетевого трафика
- TCP stream reconstruction
- Понимание протокола Telnet
