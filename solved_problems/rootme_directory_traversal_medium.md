# Отчет об эксплуатации: Directory Traversal (Orange Level/Medium)

## Цель
Получение несанкционированного доступа к конфиденциальным файлам сервера путем манипуляции путями в параметрах URL.

## Исследование (Reconnaissance)
При анализе веб-приложения был обнаружен динамический параметр:
`?galerie=categories`

**Гипотеза:** Приложение использует значение этого параметра для прямого обращения к папкам на диске сервера без должной фильтрации.

## Ход эксплуатации (Steps to Reproduce)

1. **Анализ структуры:** При удалении значения параметра было замечено изменение содержимого страницы, что подтвердило возможность управления отображением директорий.
2. **Инспекция кода:** С помощью *DevTools* (Инструменты разработчика) в браузере был исследован HTML-код страницы. 
3. **Поиск скрытых данных:** - Обнаружен скрытый элемент с некорректно отображаемой картинкой.
   - Атрибут `src` указывал на файл с подозрительным именем (например, `86hwnX2r`).
4. **Обход ограничений:** Подстановка найденного имени в параметр `?galerie=` позволила войти в закрытую директорию.
5. **Финализация:** Внутри папки был найден файл `password.txt`. Прямое обращение к нему через URL раскрыло содержимое флага.

## Результат
Уязвимость подтверждена. Получен полный доступ к файлу с паролем.
